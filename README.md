## ⚖️ Part 5: GAN vs Diffusion (Through Optimization Lens)

* **Chapter 22:** [Core Differences in Learning Behavior](https://github.com/fromsantanu/GenTechTut/blob/main/Part5/p01.md)

| Aspect           | GAN                              | Diffusion                 |
| ---------------- | -------------------------------- | ------------------------- |
| Loss             | Dual (Generator + Discriminator) | Single (Noise prediction) |
| Optimization     | Competitive (minimax)            | Direct minimization       |
| Stability        | Often unstable                   | Highly stable             |
| Convergence      | Oscillatory                      | Smooth                    |
| Output diversity | Can collapse                     | Good diversity            |
