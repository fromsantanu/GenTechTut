# 📘 Step 3: Define Noise Addition Function

We simulate the forward diffusion process.

---

# 🌫️ Add Gaussian Noise

```python id="4pczwt"
def add_noise(images, noise_level=0.5):

    noise = torch.randn_like(images)

    noisy_images = images + noise_level * noise

    noisy_images = torch.clamp(noisy_images, 0., 1.)

    return noisy_images, noise
```

---

# 🧠 What This Function Does

| Step                  | Purpose            |
| --------------------- | ------------------ |
| Generate random noise | Simulate diffusion |
| Add noise to image    | Create noisy input |
| Clamp values          | Keep pixels valid  |

# 🧱 Denoising Network

```python id="mygxrg"
class Denoiser(nn.Module):

    def __init__(self):
        super().__init__()

        self.encoder = nn.Sequential(
            nn.Conv2d(1, 32, 3, padding=1),
            nn.ReLU(),

            nn.Conv2d(32, 64, 3, padding=1),
            nn.ReLU()
        )

        self.decoder = nn.Sequential(
            nn.Conv2d(64, 32, 3, padding=1),
            nn.ReLU(),

            nn.Conv2d(32, 1, 3, padding=1)
        )

    def forward(self, x):

        x = self.encoder(x)

        x = self.decoder(x)

        return x
```

---

# 🧠 What the Network Learns

The model learns:

> How noisy patterns differ from meaningful image structure.

# 📘 Step 5: Create Model and Optimizer

```python id="0vf9jz"
model = Denoiser()

criterion = nn.MSELoss()

optimizer = optim.Adam(model.parameters(), lr=0.001)
```

---

# 🧠 Why MSE Loss Is Used

Diffusion models commonly predict:

* Noise values

MSE measures:

* Difference between actual and predicted noise

---

# 📘 Step 6: Training Loop

This is the learning stage.

---

# 🔄 Training Process

For every batch:

1. Add noise
2. Predict noise
3. Calculate loss
4. Update weights

---

# 📘 Full Training Code

```python id="0wzc8d"
epochs = 5

for epoch in range(epochs):

    for images, _ in loader:

        noisy_images, actual_noise = add_noise(images)

        predicted_noise = model(noisy_images)

        loss = criterion(predicted_noise, actual_noise)

        optimizer.zero_grad()

        loss.backward()

        optimizer.step()

    print(f"Epoch {epoch+1}, Loss: {loss.item():.4f}")
```

---

# 🧠 Important Insight

The model is not directly predicting:

* Clean image

It predicts:

* Noise

This is the key idea behind diffusion learning.

```python id="35wjlwm"
images, _ = next(iter(loader))

noisy_images, noise = add_noise(images)

predicted_noise = model(noisy_images)

denoised_images = noisy_images - predicted_noise

fig, axes = plt.subplots(3, 6, figsize=(10,5))

for i in range(6):

    axes[0,i].imshow(images[i][0], cmap='gray')
    axes[0,i].axis('off')

    axes[1,i].axis('off')

    axes[2,i].imshow(denoised_images[i][0].detach(), cmap='gray')
    axes[2,i].axis('off')

axes[0,0].set_title("Original")
axes[1,0].set_title("Noisy")

plt.show()
```
